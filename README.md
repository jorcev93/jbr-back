#JBRE API
1. Clonar el proyecto
3. Ejecutar `yarn install` para instalar las dependencias
3. Clonar el archivo `.env.template` y renombrarlo a `.env`
4. Agregar las variables de entorno al archivo `.env`
5. Ejecutar `docker compose up -d` para levantar la base de datos
4. Ejecutar `yarn start:dev` para iniciar el servidor en modo desarrollo
5. Ejecutar `npm run start:prod` para iniciar el servidor en modo producci√≥n
